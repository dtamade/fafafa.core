## 技术发现

### Parking Lot Mutex 实现特点

1. **多层回退策略**
   - Windows 8.1+: 使用 WaitOnAddress/WakeByAddressSingle
   - Vista/7: 使用 NTDLL Keyed Events
   - XP: 使用信号量 + 等待者计数

2. **状态管理**
   - 使用 32 位原子整数存储状态
   - LOCKED_BIT (0x01): 锁定位
   - PARKED_BIT (0x02): 有线程等待位
   - FALLBACK_BIT (0x04): 回退模式位

3. **性能优化**
   - 快速路径：单次 CAS 操作
   - 智能自旋：MAX_SPIN_COUNT = 40
   - 退避策略：SPIN_BACKOFF_THRESHOLD = 10
