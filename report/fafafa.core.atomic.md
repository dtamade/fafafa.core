## 遇到的问题与解决方案

### 1. 测试挂起问题
**问题**：早期测试中出现无限循环，导致测试挂起
**原因**：CAS 自旋没有上限，三线程内存序测试无界轮询
**解决**：
- 所有 CAS 自旋加上限（100万次）+ Yield + 超时抛异常
- 内存序测试改为有界轮询（10万次）+ Yield
- 统一线程生命周期管理（FreeOnTerminate=False）

### 2. 指针桥接警告
**问题**：大量 Hint(4055) "Conversion between ordinals and pointers"
**原因**：指针与整数间的类型转换
**解决**：保留警告（不影响功能），在文档中说明这是有意的设计

### 3. 64位常量范围检查
**问题**：某些64位常量触发 Range check 错误
**原因**：FPC 对有符号/无符号常量的处理
**解决**：使用 QWord() 包装，避免编译器误判

### 4. 平台兼容性
**问题**：不同平台的原子操作 API 差异
**解决**：
- 优先使用平台原生 API（Windows Interlocked）
- 后备使用 FPC RTL 跨平台函数
- 编译时条件编译适配
