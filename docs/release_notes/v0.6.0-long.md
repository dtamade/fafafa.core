Compatibility
- Facade is a thin layer; fixed layer behavior remains unchanged
- Object ForEach allocates a transient String for the key; values are zero‑copy wrappers
- Recommended migrations:
  - Replace exception-throwing Get* with TryGet/OrDefault where tolerant behavior is desired
  - Replace manual loops with JsonArrayForEach / JsonObjectForEach for clarity and early-stop
  - Prefer JsonPointerGet for read-only path access
