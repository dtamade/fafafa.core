# YAML 模块支持特性矩阵（当前实现快照）

特性覆盖以“可用/部分/未支持”简表，持续更新。

| 类别 | 特性 | 状态 | 说明 |
|---|---|---|---|
| 事件 | 流/文档起止 | 可用 | STREAM/DOCUMENT START/END |
| 事件 | 标量 | 可用 | 去尾空白、裁剪注释，保留键前导空白 |
| 事件 | Flow 映射 `{}` | 可用 | 支持键/值引号、逗号分隔（容忍多逗号）；值可为嵌套 flow 映射/序列 |
| 事件 | Flow 序列 `[]` | 可用 | 支持逗号分隔（容忍多逗号）；序列项可为标量或内嵌 flow 映射；可与 flow 映射混合嵌套 |
| 事件 | 别名/锚点/标签 | 部分 | tokenizer 占位，解析路径待扩展 |
| 非 Flow | 多行映射 | 部分 | 已覆盖关键场景，仍在增强 |
| 错误处理 | 不合法关闭符 | 部分 | 有限容错，待完善诊断 |
| 文档模型 | 构建/访问 | 占位 | API 已定义，实现后续迭代 |
| 发射器 | 输出 | 占位 | API 已定义，实现后续迭代 |

注：本表反映当前测试覆盖通过的能力，不等同完整 YAML 1.2 规范。
