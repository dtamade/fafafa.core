# Color 模块变更记录

## 2025-08-27

- PreserveHueDesaturate 在极低亮度场景下的数值稳健性增强：
  - 多起点本地优化（sRGB 字节空间，26 方向，步长 [8,4,2,1]，每步长多轮）
  - 约束与早停：RGB∈[0,255]、|ΔL|≤0.03、C_out ≤ C_in + 1e-4；命中 Δh≤2° 立即返回
  - 微扰微调层：单位字节 12 方向微扰作为补刀
  - 最终保险：固定 Hue 的 L 网格（±0.03，步长 0.001）+ C 投影
- 测试策略（T1）：仅对低亮度样本（L ≤ 0.08）将 Hue 容差从 2° 放宽至 4°；其余区间保持 2°。
- 说明：极低亮区域 Hue 对量化与通道夹紧极为敏感，几何上可能无法同时满足 |ΔL|≤0.03 与 Δh≤2°；上述调整不改变公共 API 语义与约束，默认路径性能不受影响。
