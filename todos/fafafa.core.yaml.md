## 当前任务：YAML 词法分析器实现

### 目标
移植 libfyaml 的词法分析器，实现 YAML 标记识别、字符串处理、数字解析等功能。

### 需要移植的文件
1. **fy-token.h / fy-token.c** - 标记定义和基础操作
2. **fy-atom.h / fy-atom.c** - 原子数据结构（字符串片段）
3. **fy-input.h / fy-input.c** - 输入处理和缓冲
4. **fy-scan.c** - 词法扫描器核心逻辑

### 技术风险
- **复杂度高**：词法分析器逻辑复杂
  - 缓解：分阶段实现，逐步验证
- **性能要求**：需要高效的字符串处理
  - 缓解：使用内存池和零拷贝技术
- **兼容性**：与 libfyaml 行为一致
  - 缓解：详细的对比测试

### 进度风险
- **时间估算**：可能低估实现复杂度

- **依赖关系**：后续模块依赖词法分析器
  - 缓解：确保接口稳定，优先实现核心功能
